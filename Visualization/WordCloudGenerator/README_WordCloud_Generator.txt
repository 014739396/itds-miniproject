1. Download https://www.kaggle.com/Cornell-University/movie-dialog-corpus. Unzip it
2. Go to the folder which contains the unzipped folder movie-dialog-corpus
3. Prerequisited for WordCloud:
   First install http://landinghub.visualstudio.com/visual-cpp-build-tools for WordCloud
   Pip install WordCloud 
4. %run WordCloudGenerator_GUI.py
The fourth step first cleans the dialogues.
After a while, you should get a GUI, where you can select the movie name from drop down list. Then click on 'Please give options for characters'. Choose the character name from drop down list. Then click on Generate WordCloud.
